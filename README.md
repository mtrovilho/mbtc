mbtc
====

CLI para [Mercado Bitcoin](http://www.mercadobitcoin.com.br/)


Installation
------------

Add this line to your application's Gemfile:

    gem 'mbtc'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mbtc
